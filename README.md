# Fraud Game (Python)

Консольная игра в стиле **Rogue (1980)** с генерацией
подземелий, реализованная на **Python 3.12**.\

Игра полностью пошаговая: каждое действие игрока запускает действия
противников.

------------------------------------------------------------------------

# Особенности проекта

-   21 случайным оразом сгенерированный уровень подземелий
-   9 комнат на каждом уровне
-   Комнаты соединены коридорами (гарантированная связность)
-   Пошаговая система боя
-   6 типов уникальных противников с разным поведением
-   Инвентарь и система предметов
-   Туман войны (Fog of War)
-   Система статистики и таблица рекордов
-   Сохранение и загрузка игрового прогресса (JSON)
-   Консольный интерфейс

### Domain

Содержит **всю игровую логику**:

-   генерацию уровней
-   систему боя
-   поведение монстров
-   управление персонажем
-   работу предметов
-   правила игры

### Core

Отвечает за:

-   отрисовку карты
-   отображение интерфейса

Отвечает за:

-   сохранение прогресса
-   загрузку игровой сессии
-   таблицу рекордов

Формат хранения --- **JSON**.

# Игровые сущности

## Игровая сессия

Хранит:

-   текущий уровень
-   персонажа
-   статистику прохождения
-   прогресс игрока

## Уровень

Содержит:

-   9 комнат
-   коридоры
-   противников
-   предметы
-   точку входа
-   точку выхода

## Персонаж

Характеристики:

-   `health`      - здоровье
-   `max_health`  - максимальное здоровье
-   `dexterity`   - ловкость
-   `strength`    - сила
-   `weapon`      - оружие

Дополнительно:

-   рюкзак
-   количество сокровищ
-   статистика действий

## Рюкзак

Хранит:

-   еду
-   эликсиры
-   свитки
-   оружие
-   сокровища

Ограничения:

-   максимум **9 предметов каждого типа**

## Противники

Типы врагов:

  Тип          Символ   Особенности
  ------------ -------- --------------------------------
  Zombie       Z        высокий HP  
  Vampire      V        первый удар по нему --- промах  
  Ghost        G        телепортация  
  Ogre         O        сильные атаки  
  Snake        S        диагональное движение  

Каждый враг имеет:

-   здоровье
-   силу
-   ловкость
-   враждебность

------------------------------------------------------------------------

# Система предметов

Типы предметов:

### Сокровища

-   выпадают из монстров и увеличивают итоговый счет

### Еда

-   восстанавливает здоровье

### Эликсиры

-   временно увеличивают характеристики

### Свитки

-   **постоянно** увеличивают характеристики

### Оружие

-   влияет на формулу урона, наносимого противнику

------------------------------------------------------------------------

# Система боя

Бой происходит в **3 этапа**:

1.  Проверка попадания
2.  Расчет урона
3.  Применение урона

Формула попадания зависит от:

-   ловкости атакующего
-   ловкости цели

Формула урона зависит от:

-   силы
-   оружия

------------------------------------------------------------------------

# Генерация уровней

Каждый уровень:

-   содержит 9 комнат случайного размера
-   комнаты соединены коридорами
-   граф комнат всегда **связный**

Гарантируется:

-   одна стартовая комната
-   одна конечная комната

------------------------------------------------------------------------

# Fog of War

Используется система тумана войны:

-   неизвестные комнаты не отображаются
-   посещенные комнаты показывают только стены
-   текущая комната полностью видна

Видимость рассчитывается с помощью:

-   **Ray Casting**
-   **алгоритма Брезенхэма**

------------------------------------------------------------------------

# Управление

  Клавиша   Действие
  --------- ---------------------
  w         вверх
  a         влево
  s         вниз
  d         вправо
  h         использовать оружие
  j         съесть еду
  k         выпить эликсир
  u         прочитать свиток
  m         открыть дверь ключом

При использовании предметов появляется список:

    Выберите предмет (1–9)

------------------------------------------------------------------------

# Статистика

В игре собирается статистика:

-   количество сокровищ
-   достигнутый уровень
-   побежденные противники
-   съеденная еда
-   выпитые эликсиры
-   прочитанные свитки
-   нанесенные удары
-   полученные удары
-   пройденные клетки

Все попытки сохраняются и отображаются в **таблице рекордов**.

------------------------------------------------------------------------

# Сохранение игры

Прогресс сохраняется в **JSON**.

Сохраняется:

-   текущий уровень
-   состояние игрока
-   карта уровня
-   позиции всех сущностей
-   статистика

После перезапуска можно продолжить последнюю сессию.
